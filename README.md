Aplikacja webowa z wykorzystaniem bibloteki p5.js do podstawowego przetwarzania obrazów.

**Funkcje:**
- histogram (Gray + kanały RGB),
- operacje punktowe (clamp, jasność, kontrast, negatyw, progowanie, gamma),
- operacje na dwóch obrazach (dodawanie, różnica, średnia, itp.),
- filtracja liniowa (konwolucja z maską użytkownika + presety),
- filtracja nieliniowa (min/max/mediana/ważona mediana),
- wykrywanie krawędzi (Sobel, Prewitt, Roberts),
- morfologia matematyczna (erozja, dylatacja).
